# Week 1 - Hands-on Exercises

This repository contains the Week 1 exercises covering Design Patterns, Data Structures & Algorithms, PL/SQL Programming, Test-Driven Development, Mockito, and SLF4J Logging.

## Contents

### 1. Design Patterns and Principles
- Implementing the Singleton Pattern
- Implementing the Factory Method Pattern

### 2. Data Structures and Algorithms
- E-Commerce Platform Search Function
- Financial Forecasting

### 3. PL/SQL Programming
- Control Structures
- Stored Procedures

### 4. Test-Driven Development and Logging Framework

#### JUnit – Basic Testing
- Setting Up JUnit
- Assertions in JUnit
- Arrange–Act–Assert (AAA) Pattern
- Test Fixtures
- Setup and Teardown Methods

#### Mockito
- Mocking and Stubbing
- Verifying Interactions

#### SLF4J Logging
- Logging Error Messages and Warning Levels

## Technologies Used
- Java
- Maven
- JUnit 5
- Mockito
- SLF4J
- Spring Tooling (Eclipse)
- Oracle PL/SQL
